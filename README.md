# Pype-ERM

A modern, multi-tenant Education Resource Management (ERM) system built with React, TypeScript, Node.js, Express, Prisma, and PostgreSQL.

## Features

- **Multi-Tenant Architecture**: Robust tenant separation and custom branding.
- **Role-Based Access Control (RBAC)**: Secure access for Super Admins, Organization Admins, CEOs, Department Admins, and Employees.
- **Enterprise Ready**: Seamlessly integrated department architectures, scalable license management, and secure JWT-based session controls.
- **Modern Stack**:
  - **Frontend**: React 19, TypeScript, Vite, TailwindCSS, Radix UI.
  - **Backend**: Node.js, Express, Prisma ORM, PostgreSQL.
